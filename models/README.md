# models

Esta pasta conterá os models do projeto — funções que interagem diretamente com o banco de dados (queries SQL).

Serão criados nas próximas etapas:
- `Cardapio.js`   → queries de categorias e itens do menu
- `Pedido.js`     → queries de criação e atualização de pedidos
- `Usuario.js`    → queries de autenticação de usuários admin
