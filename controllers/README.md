# controllers

Esta pasta conterá os controllers do projeto — funções que processam as requisições e retornam respostas.

Serão criados nas próximas etapas:
- `cardapioController.js`  → lógica do cardápio online
- `pedidosController.js`   → lógica de criação e listagem de pedidos
- `adminController.js`     → lógica do painel administrativo
- `authController.js`      → lógica de autenticação (login JWT)
